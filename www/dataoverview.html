Data overview

At this point, a dataset recorded from 20 participants is publicly available. For each participant a number of different scans and auxilliary recordings have been obtained and several types of minimally preprocessed data are provided in addition. The resulting dataset is several hundreds of gigabytes large. This pages provides a short overview of the dataset in order to aid selection. The full description of the data release is available in a dedicated publication.
